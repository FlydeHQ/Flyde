# Flyde Example Project - Hello World

"Hello World" using Flyde. Check out the [step-by-step tutorial here](https://www.flyde.dev/docs/hello-world-with-flyde/) for more info.

![Preview](preview.gif)

## Prerequisites

1. [VS Code](https://code.visualstudio.com/)
2. [Flyde VS Code Extension](https://marketplace.visualstudio.com/items?itemName=flyde.flyde-vscode)
3. Yarn

## Running locally

1. Open "HelloFlyde.flyde" in VSCode
2. `node index.js` (while watching the Flyde editor in VSCode!) 

Looking to learn more about Flyde? Visit the official website at https://www.flyde.dev
